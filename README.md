# E-Commerce Sales & Customer Behavior Analytics Dashboard

## Overview

This project is an interactive Excel dashboard built using an Indian E-Commerce Sales & Customer Behavior dataset containing 25,000+ records.

The dashboard provides business insights related to:

* Revenue trends
* Customer behavior
* Conversion analysis
* Cart abandonment analysis
* Product category performance
* Device-wise revenue analysis
* Payment method distribution
* Marketing channel performance

* Total Revenue
* Total Customers

* Average Rating
* Conversion Rate
* Cart Abandonment Rate

---

## Business Insights

* Mobile users generated the highest revenue.
* Home & Kitchen category contributed maximum sales.
* UPI and Credit Card were the most preferred payment methods.
* Returning users showed better conversion behavior.

---

## Business Recommendations

* Focus More on Mobile Users.
* Reduce Cart Abandonment.
* Promote High-Performing Categories.
* Optimize Marketing Channels.
* Improve Payment Flexibility.
